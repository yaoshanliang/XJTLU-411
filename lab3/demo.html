Listeners

Keyboard listener

To add a keyboard listener, add the event listener to your script. Of the 3 parameters, the first one is the property to listen out for "keydown", i.e. when the key is pressed down, the second one is the function that is called when a key is detected, and the third one, just keep as "false".

To check the key, create a function matching the name, and check for key value

Mouse listener

To add a mouse listener, you first need to use the query to search for the element to assign a listener to, and add a listener. Then create a function to match, and this function will be called when your chosen element is clicked on.
